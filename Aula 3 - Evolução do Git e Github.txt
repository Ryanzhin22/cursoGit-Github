Quem veio primeiro o Git ou o GitHub?
-> Pra responder isso é necessário um pouco história...

1985 - CVS
-> Centralizado
-> Mais Popular
-> Open Source
-> Alguns problemas (consistência e velocidade)

2000 - SVN
-> Centralizado
-> Open Source
-> Ativo até hoje
-> CVS-like (queria ser um CVS)

2000 - BitKeeper
-> Distribuido
-> Proprietário
-> Versão comunidade
-> CVS-free (queria ser diferente do CVS e SVN)

2004 - BitKeeper (Problema)
-> SourcePuller
-> Engenharia Reversa
-> Recursos destravados através de "crack"
-> Recursos pagos foram destravados na versão da comunidade

2005 - BitKeeper
-> Nova licença
-> Bloqueava acesso a metadados
-> Só poderia ver versões antigas na versão comercial (contradição)
-> Linus Torvalds se revolta

2005 - Git
-> Linus Torvalds cria o Git
-> Distribuido
-> Open Source
-> Feito em 10 dias
-> Focado em Performance (CVS: 3 minutos, SVN: 1,5 minuto, GIT: 2 segundos)

2008 - GitHub
-> Proprietário
-> Hospedagem de código
-> Baseado em Git
2009: 46k repositórios
2010: 100k usuários / 1M repositórios
2011: Ultrapassou SourceForge
2013: 3M usuários / 5M repositórios
2016: 14ºlugar na lista da Forbes
2018: Maior ataque de DDoS da história

2018 - GitHub
-> Comprados pela Microsoft
-> US$ 7.5 Bilhões
-> Microsoft coloca seus próprios projetos no GitHub (PowerShell, VSCode, Chakra)
-> Operação Independente
2020: Github compra a NPM (Gestor de Pacote)
