SEGURANÇA NO GITHUB

-------------------------------------------------

SENHAS
-> Pelo menos 8 caracteres
-> Letras e números
-> Maiúsculas e minúsculas
-> Símbolos
-> Evitar nomes e palavras comuns
-> Evitar padrões
-> Nunca compartilhe uma senha

------------------------------------------------

Hábitos seguros
-> Ativar 2FA
-> Não usar tokens de recuperação de outras redes sociais
